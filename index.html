What's actually in the box

Not a wrapper. A real interpreter.

Lang isn't Python with a new file extension — it's a lexer, a recursive-descent parser, and a tree-walking evaluator, written from scratch so every part of it is readable in an afternoon.

Aa

A real pipeline

Source text → tokens → AST → evaluated. No cheating by shelling out to another language.

ƒ

Functions that close

Functions capture their defining scope properly — closures and counters work the way you'd expect.

{ }

One dict, one syntax

Every keyword is defined in a single dictionary. Rename let to var in one line.

⬢

Open, hackable, yours

MIT licensed, pip-installable, and small enough to actually read start to finish.

Get running

Four steps to your first program

No dependencies beyond Python 3.8+. Clone it, install it, run it.

  1. Download and unzip the source 
    unzip project-lang.zip
cd Project_Lang
  2. Install it as an editable package 
    pip install -e .
  3. Run one of the bundled examples 
    lang examples/fizzbuzz.lang
  4. Or drop straight into the REPL 
    lang

A taste of the syntax

Reads the way it runs

Braces for blocks, semicolons for statements, parens around conditions. Nothing hidden, nothing inferred.

Conditionals
let grade = 85;

if (grade >= 90) {
    print("A");
} elif (grade >= 80) {
    print("B");
} else {
    print("C or below");
}
// B
Closures
func make_counter() {
    let count = 0;
    func increment() {
        count = count + 1;
        return count;
    }
    return increment;
}

let counter = make_counter();
print(counter());
print(counter());
// 1
// 2
