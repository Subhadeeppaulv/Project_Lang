Documentation

A complete guide to writing programs in Lang: syntax, built-in functions, and plenty of worked examples.

1. The absolute basics

Three rules govern every Lang program:

  1. Every statement ends in a semicolon ;
  2. Every block of code is wrapped in curly braces { }
  3. Every condition (in if, elif, while) is wrapped in parentheses ( )
let x = 5;
if (x > 0) {
    print("positive");
}

Run a file with:

lang myfile.lang

2. Comments

Anything after # on a line is ignored:

# This is a comment
let x = 5;  # this is also a comment

3. Variables

Declare a variable with let. Reassign it later with just = (no let the second time).

let name = "Alice";
let age = 30;

age = 31;              # reassignment
print(name, age);      # Alice 31

Data types

TypeExamples
Number42, 3.14, -7
String"hello", "line\nbreak"
Booleantrue, false

String escape sequences: \n (newline), \t (tab), \" (quote), \\ (backslash).

4. Operators

Arithmetic

print(5 + 3);   # 8
print(5 - 3);   # 2
print(5 * 3);   # 15
print(5 / 3);   # 1.6666666666666667
print(5 % 3);   # 2  (remainder)

Comparison

All comparison operators return true or false.

print(5 > 3);    # true
print(5 < 3);    # false
print(5 >= 5);   # true
print(5 <= 4);   # false
print(5 == 5);   # true
print(5 != 3);   # true

Logical

print(true and false);   # false
print(true or false);    # true
print(not true);         # false

String concatenation

+ joins strings, and automatically converts numbers to strings:

let name = "World";
print("Hello, " + name + "!");     # Hello, World!
print("Score: " + 42);             # Score: 42

5. Printing / output

print takes any number of arguments, separated by commas, and prints them space-separated:

print("x =", 5);          # x = 5
print(1, 2, 3);           # 1 2 3
print("just one thing");  # just one thing

7. Conditionals — if / elif / else

let age = 20;

if (age < 13) {
    print("child");
} elif (age < 20) {
    print("teenager");
} else {
    print("adult");
}
# adult

elif can repeat as many times as you need, and else is optional:

let grade = 85;

if (grade >= 90) {
    print("A");
} elif (grade >= 80) {
    print("B");
} elif (grade >= 70) {
    print("C");
} else {
    print("F");
}
# B

A minimal if with no else:

let x = 5;
if (x > 0) {
    print("positive");
}

8. Loops — while

Lang currently has one loop construct: while. It repeats a block as long as its condition is true.

Basic counting loop

let i = 0;
while (i < 5) {
    print(i);
    i = i + 1;
}
# 0
# 1
# 2
# 3
# 4

Counting down

let i = 10;
while (i > 0) {
    print(i);
    i = i - 1;
}
print("Liftoff!");

Loop with a condition inside

let i = 0;
while (i < 10) {
    if (i % 2 == 0) {
        print(i);
    }
    i = i + 1;
}
# prints 0, 2, 4, 6, 8

Nested loops (multiplication table)

let i = 1;
while (i <= 3) {
    let j = 1;
    while (j <= 3) {
        print(i, "x", j, "=", i * j);
        j = j + 1;
    }
    i = i + 1;
}

Accumulator pattern (sum of 1 to 10)

let total = 0;
let i = 1;
while (i <= 10) {
    total = total + i;
    i = i + 1;
}
print(total);   # 55

9. Functions

Define a function with func, call it like you'd expect, return a value with return.

func add(a, b) {
    return a + b;
}

print(add(3, 4));   # 7

No return value

A function with no return implicitly returns nil.

func greet(name) {
    print("Hello, " + name + "!");
}

greet("Alice");   # Hello, Alice!

No parameters

func say_hi() {
    print("Hi there!");
}

say_hi();

Recursion — factorial

func factorial(n) {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

print(factorial(5));   # 120

Recursion — Fibonacci

func fib(n) {
    if (n < 2) {
        return n;
    }
    return fib(n - 1) + fib(n - 2);
}

let i = 0;
while (i < 10) {
    print(fib(i));
    i = i + 1;
}

Early return

func is_even(n) {
    if (n % 2 == 0) {
        return true;
    }
    return false;
}

print(is_even(4));   # true
print(is_even(7));   # false

10. Closures

A function defined inside another function "remembers" the variables from where it was created, even after the outer function has finished running.

func make_counter() {
    let count = 0;
    func increment() {
        count = count + 1;
        return count;
    }
    return increment;
}

let counter = make_counter();
print(counter());   # 1
print(counter());   # 2
print(counter());   # 3

Each call to make_counter() creates a brand-new, independent counter:

let counter_a = make_counter();
let counter_b = make_counter();

print(counter_a());   # 1
print(counter_a());   # 2
print(counter_b());   # 1   (independent of counter_a)

11. Scope

Variables declared inside { } only exist inside that block. A variable inside a block can "shadow" (temporarily hide) a variable of the same name from an outer scope.

let x = "outer";
{
    let x = "inner";
    print(x);   # inner
}
print(x);       # outer

12. Full example programs

FizzBuzz

let i = 1;
while (i <= 30) {
    if (i % 15 == 0) {
        print("FizzBuzz");
    } elif (i % 3 == 0) {
        print("Fizz");
    } elif (i % 5 == 0) {
        print("Buzz");
    } else {
        print(i);
    }
    i = i + 1;
}

Check if a number is prime

func is_prime(n) {
    if (n < 2) {
        return false;
    }
    let i = 2;
    while (i * i <= n) {
        if (n % i == 0) {
            return false;
        }
        i = i + 1;
    }
    return true;
}

let num = 1;
while (num <= 20) {
    if (is_prime(num)) {
        print(num, "is prime");
    }
    num = num + 1;
}

Celsius to Fahrenheit

func celsius_to_fahrenheit(c) {
    return c * 9 / 5 + 32;
}

print(celsius_to_fahrenheit(0));     # 32
print(celsius_to_fahrenheit(100));   # 212
print(celsius_to_fahrenheit(37));    # 98.6

Greatest common divisor (Euclid's algorithm)

func gcd(a, b) {
    if (b == 0) {
        return a;
    }
    return gcd(b, a % b);
}

print(gcd(48, 18));   # 6

Guess comparison

func check_guess(guess, answer) {
    if (guess == answer) {
        print("Correct!");
    } elif (guess < answer) {
        print("Too low.");
    } else {
        print("Too high.");
    }
}

check_guess(7, 10);   # Too low.
check_guess(15, 10);  # Too high.
check_guess(10, 10);  # Correct!

15. Not yet supported

Lang is an early, hand-built language — these are known gaps, not bugs:

  • No lists / arrays
  • No for loop (only while)
  • No string indexing ("hello"[0] doesn't work)
  • No modules / imports
  • No classes / objects
